# OpenAI Playground

## Install

1. Install [asdf](https://asdf-vm.com/guide/getting-started.html)

2. Install NodeJS

```sh
asdf install
```

3. Set API key as environment variable

```sh
export OPENAI_API_KEY="<YOUR API KEY>"
```

## Usage Example

```sh
node gettingStarted.js
```
